1._config.yml ( Hexo主配置文件 )

作用: ( 用来控制Hexo本身 )

  • 标题
  • 语言
  • 主题名称
  • 部署到github pages的配置
  • 站点URL
  • 生成规则,分页

例如:

Hexo的配置在 _config.yml
主题的配置在 themes/fluid/_config.yml

2.package.json ( npm项目说明)

Hexo本质是一个Node.js项目

这个文件定义:

  • Hexo的依赖包
  • 版本
  • 插件
  • npm脚本(可选)

一般不需要修改,系统用来识别blog项目
